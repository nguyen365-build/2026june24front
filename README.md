# 2026june24front

A simple public image gallery website hosted on **GitHub Pages**.

## How to add your images (up to 10)

1. Put your image files in the [`images/`](images) folder.
2. Name them `img1`, `img2`, … up to `img10`.
3. Use any of these formats: `.jpg`, `.jpeg`, `.png`, `.webp`, `.gif`.
   Example: `images/img1.jpg`, `images/img2.png`, …
4. Commit and push. The site updates automatically within a minute.

Empty slots are skipped automatically — you don't need all 10.
Click any image to view it full-size.

## Live site

Once Pages finishes building, the gallery is available at the URL shown in
the repository's **Settings → Pages** (and in the repo's "About" section).
